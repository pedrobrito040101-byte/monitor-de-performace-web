# ⚡ WebPerf Studio - Monitor de Performance

Um analisador de saúde para projetos Front-End moderno e elegante, focado em métricas de performance e SEO.

## 🔗 Demonstração
O projeto está online e pode ser testado aqui:
👉 **[Visualizar Projeto ao Vivo](https://pedrobrito040101-byte.github.io/monitor-de-performace-web/)**

---

## 🚀 Sobre o Projeto
Este dashboard foi desenvolvido para fornecer um feedback rápido e visual sobre a qualidade do código e otimização de sites. Com uma interface baseada em **Glassmorphism**, o monitor simula uma análise técnica e apresenta os resultados através de gráficos circulares animados.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização avançada com efeitos de transparência e Flexbox.
- **JavaScript (Vanilla)**: Manipulação dinâmica do DOM e animações de SVG.

## 📂 Estrutura de Pastas
O projeto segue uma organização profissional:
- `/css`: Ficheiros de estilo.
- `/js`: Lógica em JavaScript.
- `index.html`: Ponto de entrada do projeto.
